Добро пожаловать на наш сайт, где вас ждёт удобство покупок и бесконечный выбор товаров. Мы изменили подход к нашим онлайн-покупкам, чтобы вы чувствовали себя комфортно и удобно.

Основные возможности:
- Адаптивный дизайн (Mobile First)
- Автоматический слайдер баннеров с пагинацией
- Корзина товаров с динамическим счетчиком
- Добавление/удаление товаров в корзину
- Поиск товаров
- Личный кабинет (заглушка)
- Современный UI/UX дизайн

Технические особенности:
- Чистый CSS без фреймворков
- Полная адаптивность под все устройства
- Оптимизированная производительность
- Семантическая верстка (SEO-friendly)
- Динамический контент через JavaScript

Frontend:
- HTML5 - семантическая структура
- CSS3 - стилизация и анимации
- JavaScript (ES6+) - интерактивность
- SVG - векторные иконки

Инструменты:
- Flexbox/Grid - построение макетов
- CSS Variables - управление стилями
- Event Delegation - обработка событий

Функциональность сайта.

Шапка сайта:
- Логотип - переход на главную страницу
- Поиск - поиск товаров по сайту (заглушка)
- Корзина - отображение количества добавленных товаров
- Личный кабинет - вход в аккаунт (заглушка)
- Бургер-меню - навигация на мобильных устройствах

Баннер:
- Автоматическое переключение слайдов (каждые 5 секунд)
- Пагинация для ручного переключения
- Остановка при наведении мыши
- Адаптивная высота под разные экраны

Карточки товаров:
- Изображение с обрезкой и центрированием
- Цена - актуальная стоимость
- Название с обрезанием длинных названий
- Рейтинг - звезды + количество отзывов
- Кнопка "добавить" - добавление в корзину с анимацией
- Визуальный отклик - изменение цвета кнопки при добавлении

Корзина:
- Подсчет общего количества товаров
- Индикатор на иконке корзины
- Добавление/удаление товаров одним кликом
- Обновление в реальном времени
